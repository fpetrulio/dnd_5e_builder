## Visione del prodotto
Tool web per la creazione e analisi di build di personaggi D&D 5e. Funzionalità principali:
- Import risorse ufficiali da API (Open5e, 5etools) + homebrew locale
- Creazione personaggio da qualsiasi livello con wizard guidato
- Level-up guidato con validazione automatica delle regole
- Revert a livello precedente con snapshot immutabili
- Timeline visuale della progressione e vista build per ogni livello
- Confronto differenziale tra livelli e tra build diverse
- Dashboard statistica: DPR, HP, CA, probabilità, radar chart
- AI Advisor per suggerimenti e ottimizzazione
- Party builder con ottimizzazione contestuale e sinergie
- Export PDF e condivisione build via link

Documentazione dettagliata: docs/FEATURES.md, docs/ROADMAP.md
