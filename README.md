# D&D 5e Character Builder

A tool for building and managing Dungeons & Dragons 5th Edition characters.

## Features

- Character creation following official D&D 5e rules
- Race and class selection with stat bonuses
- Ability score management (Standard Array, Point Buy, or dice roll)
- Equipment and inventory tracking
- Spell management for spellcasting classes
- Character sheet export

## Getting Started

Clone the repository and follow setup instructions as the project develops.

```bash
git clone https://github.com/fpetrulio/dnd_5e_builder.git
cd dnd_5e_builder
```
